# Student Enrollment Form using JSONPowerDB

## Description
This project is a Student Enrollment Form developed using HTML, JavaScript, and JSONPowerDB.
It stores and updates student records using Roll Number as the primary key.

## Benefits of JSONPowerDB
- NoSQL, schema-free database
- High performance
- REST API based
- No backend required
- Easy frontend integration

## Scope of Functionalities
- Add new student record
- Update existing student record
- Primary key validation
- Save, Update, Reset controls
